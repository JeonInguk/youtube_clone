## 유튜브 클론코딩 
React학습한것을 토대로 구현한 Youtube 클론코딩 프로젝트입니다.
Youtube API를 활용하여, 유튜브에 업로드된 영상을 검색하고, 시청할 수 있도록 구현했습니다.
